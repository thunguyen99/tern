Tern Build Status phiên bản NPM tình trạng kinh phí: duy trì hạnh phúc Đây là [Tern] [1]. Tern là một độc lập, biên tập viên độc lập phân tích JavaScript có thể được sử dụng để cải thiện tích hợp JavaScript của biên tập viên hiện có. Nhờ một nhóm hào phóng [nhà tài trợ đám đông] [2], Tern là phần mềm mã nguồn mở, theo một giấy phép MIT. Hiện tại có plugin có sẵn cho [Emacs] emacs, [Vim] [vim], [Text Sublime] [st], [Eclipse (và Java API chung)] [ ec], [Light Table] [lt], và [gedit] [gedit], và được xây dựng trong hỗ trợ trong [Chân đế] [dấu ngoặc] và [Cạnh Code] [edge_code]. Đối với tài liệu hướng dẫn thêm, xem các [trang dự án] [1] và [nhãn] [3].
